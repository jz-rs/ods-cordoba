# 🌱 ODS Córdoba

Plataforma web que acerca los Objetivos de Desarrollo Sostenible a la realidad de Córdoba, Argentina.

🔗 [ods-cordoba.vercel.app](https://ods-cordoba.vercel.app)

---

## ¿Qué es?

La mayoría de la gente no sabe qué son los ODS ni cómo aplicarlos en su ciudad. Este proyecto lo resuelve de dos formas: **informando** sobre los objetivos más relevantes para Córdoba, y **permitiendo que los vecinos reporten problemáticas ambientales** en un mapa colaborativo en tiempo real.

## Stack

Next.js · TypeScript · Supabase · Leaflet · Vercel
